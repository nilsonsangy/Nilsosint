
# YouTube Video Downloader

This tool downloads YouTube videos directly to your Desktop.

## Usage

To download a video, simply run:

```powershell
python video_downloader.py [YouTube_URL]
```
- You can provide the video URL as a command-line argument, or the script will prompt you for it.

The downloaded video will be saved to your Desktop with the original YouTube title.

---

## ⚠️ Disclaimer

This tool is intended for downloading only public domain or Creative Commons licensed videos from YouTube. Use responsibly.
